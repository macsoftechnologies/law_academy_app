# Law App

## Overview

A React Native mobile application built with Expo and Expo Router.

## Tech Stack

* React Native
* Expo
* Expo Router
* TypeScript

## Development Environment

| Tool    | Version     |
| ------- | ----------- |
| Node.js | v20.19.4    |
| npm     | v10.8.2     |
| Java    | 17.0.11 LTS |
| Git     | 2.45.1      |

## Installation

```bash
git clone <repository-url>
cd Law
npm install
```
